# Privacy Policy

Effective date: 2026-03-19

LinkedIn Profile Snapshot is designed to run entirely in your browser.

## Data handling

- The extension reads visible content from the active LinkedIn profile page only when you click the extension.
- The extension does not send page content, profile data, or clipboard data to any server.
- The extension does not store extracted profile data in a remote database or analytics service.
- The extension does not use cookies, tracking pixels, or third-party telemetry.

## Clipboard access

The extension can copy extracted data to your system clipboard when you press a copy button. That action happens locally on your device.

## Permissions

- `clipboardWrite` is used only for user-initiated copy actions.
- `https://www.linkedin.com/in/*` access is used only to read the current profile page in the active tab.

## Changes to this policy

If the extension changes in a way that affects data handling, this policy should be updated to match the new behavior.
