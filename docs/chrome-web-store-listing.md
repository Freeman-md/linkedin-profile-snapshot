## Description

LinkedIn Profile Snapshot is a lightweight Chrome extension for capturing the visible information on a LinkedIn profile page without leaving your browser tab. Open the extension on any `linkedin.com/in/*` profile to copy the canonical LinkedIn URL or the full visible profile snapshot in one click.

Main features:

- Captures the visible profile name, headline, location, summary, and canonical LinkedIn URL.
- Includes visible About text, recent activity snippets, and experience details when they are on screen.
- Copies the full extracted snapshot locally to your clipboard for reuse in notes, CRMs, or research workflows.
- Works directly from the toolbar popup with no backend, no account, and no analytics.

Notes for reviewers and users:

- The extension reads only LinkedIn profile pages under `https://www.linkedin.com/in/*`.
- It extracts only information visible in the page DOM at the time you open the popup.
- The current release targets the English-language LinkedIn interface.
