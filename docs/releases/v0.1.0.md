# LinkedIn Profile Snapshot v0.1.0

Initial public release.

## Highlights

- Popup-based Chrome extension for `linkedin.com/in/*` profile pages.
- Copies the canonical LinkedIn profile URL or the full visible profile snapshot.
- Captures visible About text, recent activity, and experience details when present.
- Includes fixture-based extractor tests, GitHub Actions CI, packaging workflow, privacy policy, and release docs.
